Orange

What?

Orange is an imperative, statically typed systems programming language that aims to be productive and powerful without all the clunkiness of C++.

Orange is open source: users are encouraged to submit pull requests, or report issues and propose features in the issue tracker.

If you want Orange syntax highlighting for Sublime Text, you can find that here.

Status

Orange is still in the design phase, so check back often for updates.

Goals

  • Support OOP
  • Compile to a native binary
  • Target desktop or ARM platforms like Arduino
  • Pointers and inline assembly
  • Smart pointers
  • Runtime-optional

Examples

  • Pointers 
extern int32 printf(char* s, ...)

# type id is a unique generic per parameter 
def swap(id a, id b) 
  temp = *a 
  *a = *b
  *b = temp 
end

int a = 3, b = 5
swap(&a, &b)
