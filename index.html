Orange

What?

Orange is an imperative, statically typed systems programming language that aims to be productive and powerful without all the clunkiness of C++. You can check out its documentation here.

Orange is open source: users are encouraged to submit pull requests, or report issues and propose features in the issue tracker.

If you want Orange syntax highlighting for Sublime Text, you can find that here.

Status

Orange is still in the design phase, so check back often for updates.

Goals

  • Support OOP
  • Compile to a native binary
  • Target desktop or ARM development boards
  • Low-level features: pointers and inline assembly
  • High-level: focus on concurrency, reference counting for memory management
  • High level features should feel part of the language, and not the API

Examples

  • Hello World
  • Pointers
  • Concurrency 
extern int32 printf(char* s, ...)
for (var i = 0; i < 10; i++)
  printf("Hello world, %d!\n", i)
end

def swap(var a, var b) 
  temp = *a 
  *a = *b
  *b = temp 
end

int a = 3, b = 5
swap(&a, &b)

do {
  var sum = 0
